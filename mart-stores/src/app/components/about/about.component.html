About us

Mart is an Open-Source ecommerce side project written with the use of languages like Java, Spring-Boot, Angular, Tailwind, Scss, Mysql, Maven, Data-Jpa and so many other resources.

It is a well secured end to end ecommerce web page that uses the latest security measures as of the publishing off this webpage.

Mart enables user to view list of well structured product and information regarding them, it also enables user to sign-up and sign-in into their respective accounts giving them the access to add products into the cart page and proceed with other features of this web application


Samuel is a Full-stack developer with years of experience using Java and Spring-Boot to developed web applications.
